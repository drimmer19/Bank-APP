Реализованные функции:

Аутентификация пользователя
Вход по логину и PIN; проверка учётных данных и очистка полей ввода.

Генерация логинов
Функция createLogIn создаёт короткие логины из ФИО (инициалы).

Отображение движений счёта
displayMovements выводит список операций с датой, типом (внесение/снятие) и суммой.
Поддержка сортировки по возрастанию/исходному порядку.

Форматирование дат операций
formatMovementDate форматирует даты относительно текущей (сегодня, вчера, N дней назад) и выводит локализованную дату/время.

Баланс и итоговые суммы
calcPrintBalance вычисляет и отображает текущий баланс счёта.
calcDisplaySum считает входящие, исходящие и итоговую сумму процентов/баланса и выводит в footer.

Переводы между счетами
Обработка перевода средств с валидацией получателя, суммы и остатка; добавление даты в movementsDates.

Заявки на кредит (loan)
Добавление положительной операции (loan) с записью даты и обновлением интерфейса.

Закрытие аккаунта
Удаление аккаунта при совпадении логина и PIN; скрытие интерфейса приложения.

Авто-выход (таймер сессии)
startLogOut реализует обратный отсчёт, обновление индикатора времени и автоматический выход при истечении времени.
Таймер перезапускается при действии пользователя (перевод/loan/вход).

Локализация даты и времени
Отображение текущей даты/времени после входа с использованием Intl.DateTimeFormat и локали браузера.

Переключение символа валюты в UI
По клику на баланс заменяется знак валюты в операциях (например, ₽ → RUB).

Утилитарные демонстрационные таймеры
Примеры использования setTimeout и setInterval и их очистка.

Общие расчёты по всем аккаунтам
Вычисление общего баланса всех аккаунтов с использованием map, flat, reduce.
